# Fotoprokat 24 - WordPress тема

## Описание
Верстка и интеграция в WordPress тестового макета. Тема включает адаптивный хедер, слайдер с управляемым контентом, настройки через админ-панель.

## Использованные технологии
- HTML5 / CSS3 / JavaScript
- WordPress 6.x
- SiteOrigin Slider (слайдер)
- All-in-One WP Migration (экспорт)

## Установка и настройка

1. Распаковать архив `fotoprokat` в `/wp-content/themes/`
2. Активировать тему в админ-панели
3. Установить плагин **SiteOrigin Widgets Bundle**
4. Настроить слайдер на главной странице:
   - Desktop: Height = `60vh`
   - Mobile: Height = `40vh`
   - Включить вертикальное центрирование

## Управление контентом

### Текст на слайдере
Редактируется через стандартный редактор страницы (SiteOrigin Slider → текстовый блок).

### Меню
Настраивается через **Внешний вид → Меню** (регион `primary`).

## Админ-доступ
- **Логин:** admin
- **Пароль:** [указан в письме]

## Экспорт
Выполнен через плагин **All-in-One WP Migration**. Файл экспорта доступен по ссылке в Google Drive.
